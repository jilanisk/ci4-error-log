# CI4 Log Viewer

## Features
- JSON structured logs
- IP allowlist
- Tail pagination
- Live auto-refresh(upcoming)
- Search & level filters(upcoming)
- Stack trace highlighting(upcoming)
- Charts & analytics(upcoming)

## Installation
composer require jilanisk/ci4-log-viewer

## Configuration
php spark logviewer:publish

## Usage
/log-viewer

## Security
.env based IP allowlist
 

## Config (.env)
LOGVIEWER_ALLOW_IPS=127.0.0.1
